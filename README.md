# MINI port scanner
A mini but fast port scanner using threading to check all the ports from 1 to 1024

## How to use:
- Download the project using terminal or dowanload it here
- go to the directory were the main.py file is
- run it by: python main.py
- write the Host ID (for example 127.0.0.1 or anything else)
- wait for the result in list
